# 🔥 Hotkey to Motospeed CK62 ⌨️

![Layout keyboard motospeed CK62](.github/motospeed-ck62.png)

## ⌨️ install AutoKey

```shell
sudo apt install autokey-gtk
```

Download folder Motospeed-ck62 and put it in the `.config` > `autokey` > `data` in home directory
